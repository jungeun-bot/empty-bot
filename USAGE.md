# 🏢 미팅룸 예약 슬랙봇 사용 가이드

---

## 커맨드 목록

### `/예약`
날짜, 시간, 참석자를 지정하여 미팅룸 또는 포커스룸을 예약하는 창을 엽니다. 외부 게스트 이메일 초대도 가능합니다.

### `/즉시예약`
현재 시각 기준으로 바로 사용 가능한 미팅룸을 조회하고, 한 번의 클릭으로 빠르게 예약합니다. (예: `/즉시예약 4`)

### `/수정`
내가 등록한 기존 예약을 날짜별로 조회하여 회의 이름, 시간 등을 수정하거나 예약 자체를 취소할 수 있습니다.

### `/신고`
회의실 이용 관련 건의사항, 불편사항을 관리자에게 익명으로 전달합니다. 유형을 선택하고 상세 내용을 작성합니다.

### `/설치`
(채널 관리자용) 현재 채널에 예약하기·수정/취소·신고 버튼이 포함된 예약 패널을 설치하고 고정합니다. 

### `@봇 멘션`
채널에서 봇을 멘션하여 자연어로 빈 회의실을 조회하거나 대화형으로 예약·수정·취소를 진행할 수 있습니다.

### `DM (다이렉트 메시지)`
봇에게 직접 메시지를 보내 채널 멘션과 동일한 자연어 예약·조회·수정 기능을 1:1로 이용할 수 있습니다.

### `/사용방법`
빈방있소 각 커맨드별 사용방법을 안내해드립니다.

## 빈방있소 사용 가이드!

🏠 *빈방있소 사용 가이드*

안녕하세요! 회의실 예약 봇 *빈방있소*가 도착했습니다.
이제 슬랙에서 바로 회의실 예약, 조회, 변경이 가능해요!


📌 *기본 예약*
`/예약` 입력 → 모달에서 예약 유형(미팅룸/포커스룸), 회의 이름, 날짜, 시작·종료 시간 선택 → 참석자 검색 후 추가 → 완료!
외부 게스트가 있다면 이메일 입력 시 구글 캘린더 초대장이 자동 발송됩니다.


⚡ *즉시 예약 (급한 회의)*
`/즉시예약 인원수` 입력 → 지금 바로 사용 가능한 회의실 목록이 표시됩니다.
예) `/즉시예약 4`


✏️ *예약 변경/취소*
`/수정` 입력 → 날짜 선택 → 내 예약 조회 → 수정 또는 취소


🚨 *회의실 불편사항 신고*
`/신고` 입력 → 내용 작성 시 담당자에게 바로 전달됩니다.


💬 *자연어로도 가능!*
커맨드 대신 빈방있소를 멘션하거나 DM으로 자연스럽게 말해도 돼요.
예) `@빈방있소 내일 오후 2시 4명 미팅룸 예약해줘`


궁금한 점이 있으면 빈방있소에게 DM으로 물어보세요! 🙌
