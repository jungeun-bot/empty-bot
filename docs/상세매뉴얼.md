## 1. 회의실 안내

### 미팅룸 (다인 회의용)

| 이름 | 수용인원 | 용도 |
|------|---------|------|
| **L1** | 최대 12명 | 대규모 회의, 워크숍 |
| **L2** | 최대 16명 | 대규모 회의, 세미나 |
| **M1** | 최대 5명 | 소규모 회의 |
| **M2** | 최대 5명 | 소규모 회의 |

### 포커스룸 (1인 집중 업무용)

| 이름 | 수용인원 |
|------|---------|
| **Focus 3** ~ **Focus 9** | 각 1명 (총 7개) |

### 예약 가능 시간

- **시작 시간**: 08:00 ~ 19:30 (30분 단위)
- **종료 시간**: 08:30 ~ 20:00 (30분 단위)

## 2. 기본 예약 (/예약)

가장 기본적인 회의실 예약 방법입니다.

### 사용 방법

1. Slack에서 `/예약` 입력 후 Enter
2. 예약 모달이 열립니다
3. 아래 항목을 입력합니다:

| 항목 | 설명 | 필수 |
|------|------|------|
| **예약 유형** | 미팅룸 또는 포커스룸 선택 | O |
| **회의 이름** | 회의 제목 (캘린더에 표시됨) | X |
| **날짜** | 예약할 날짜 선택 | O |
| **시작 시간** | 회의 시작 시간 | O |
| **종료 시간** | 회의 종료 시간 | O |
| **참석자** | 이름 검색 후 선택 (3글자 이상 입력) | X |
| **외부 게스트** | 외부인 이메일 (쉼표로 구분) | X |

4. 입력 완료 후 **제출** 클릭
5. 조건에 맞는 빈 회의실 목록이 표시됩니다
6. 원하는 회의실을 선택하면 **예약 완료!**

### 알아두기

- 참석자 수에 맞는 회의실만 자동으로 필터링됩니다
- 예약이 완료되면 참석자 전원에게 **구글 캘린더 초대장**이 발송됩니다
- 캘린더 이벤트 제목에 `[L1] 회의명` 형태로 회의실 이름이 표시됩니다
- 외부 게스트 이메일을 입력하면 외부인에게도 초대장이 발송됩니다
- 사용자그룹(@팀이름)을 선택하면 해당 그룹 전체 멤버가 참석자로 추가됩니다

## 3. 즉시 예약 (/즉시예약)

급하게 회의실이 필요할 때 사용합니다.

### 사용 방법

1. `/즉시예약 인원수` 입력 (예: `/즉시예약 4`)
2. 현재 시각 기준으로 **바로 사용 가능한** 회의실 목록이 표시됩니다
3. 원하는 회의실의 **예약** 버튼 클릭
4. 사용 시간, 회의 이름, 참석자를 입력
5. **예약 완료!**

### 알아두기

- 인원수를 입력하지 않으면 전체 회의실이 표시됩니다
- 각 회의실 옆에 **다음 예약까지 남은 시간**이 표시됩니다
- 포커스룸도 함께 표시됩니다

## 4. 예약 수정/취소 (/수정)

내가 만든 예약을 수정하거나 취소할 수 있습니다.

### 예약 수정

1. `/수정` 입력 후 Enter
2. 날짜를 선택합니다
3. 해당 날짜의 **내 예약 목록**이 표시됩니다
4. 수정할 예약의 **수정** 버튼 클릭
5. 변경할 항목을 수정 후 제출

### 예약 취소

1. `/수정` 입력 후 Enter
2. 날짜를 선택합니다
3. 취소할 예약의 **취소** 버튼 클릭
4. 확인 후 예약이 삭제됩니다

### 알아두기

- 본인이 만든 예약만 수정/취소할 수 있습니다
- 이미 지난 시간의 예약은 수정할 수 없습니다
- 반복 이벤트(정기 회의)는 개별 수정이 불가합니다 — 캘린더에서 직접 수정해주세요

## 5. 정기 회의 (/정기회의)

매주, 격주, 매일, 매월 반복되는 회의를 한 번에 예약합니다.

### 사용 방법

1. `/정기회의` 입력 후 Enter
2. 아래 항목을 입력합니다:

| 항목 | 설명 |
|------|------|
| **예약 유형** | 미팅룸 또는 포커스룸 |
| **회의 이름** | 정기 회의 제목 |
| **첫 회의 날짜** | 반복 시작일 |
| **시작/종료 시간** | 매 회의의 시간 |
| **반복 주기** | 매주 / 격주 / 매일(평일) / 매월 |
| **반복 요일** | 매주/격주일 때 요일 선택 |
| **반복 종료일** | 언제까지 반복할지 |
| **참석자** | 이름 검색 후 선택 |
| **외부 게스트** | 있음/없음 선택 후 이메일 입력 |

3. **예약하기** 클릭
4. 자동으로 빈 회의실을 찾아 예약합니다

### 알아두기

- 구글 캘린더에 반복 이벤트로 생성됩니다
- 반복 종료일이 지나면 더 이상 예약이 생성되지 않습니다

## 6. 예약 현황 조회

특정 날짜의 회의실 예약 현황을 확인합니다.

### 사용 방법 (패널 버튼)

1. 채널에 설치된 예약 패널에서 **📊 현황** 버튼 클릭
2. 조회할 날짜를 선택합니다
3. 회의실을 선택합니다 (전체 또는 특정 회의실)
4. **조회** 클릭

### 표시 정보

- 각 시간대별 예약된 회의 목록
- 회의 이름, 시간, 예약자 정보
- 빈 시간대 확인 가능

## 7. 불편사항 신고 (/신고)

회의실 이용 중 불편한 점이나 건의사항을 관리자에게 전달합니다.

### 사용 방법

1. `/신고` 입력 후 Enter
2. 신고 유형을 선택합니다 (시설 문제, 소음, 청결 등)
3. 상세 내용을 작성합니다
4. **제출** 클릭

### 알아두기

- 신고 내용은 관리자에게 전달됩니다

## 8. 자연어 예약 (멘션/DM)

커맨드 대신 자연스러운 말로도 예약할 수 있습니다.

### 방법 1: 채널에서 멘션

```
@빈방있소 내일 오후 2시 4명 미팅룸 예약해줘
```

### 방법 2: DM으로 대화

봇에게 직접 메시지를 보내면 1:1로 안내받을 수 있습니다.

### 자연어 예시

| 입력 | 동작 |
|------|------|
| `내일 오후 2시 4명 미팅룸` | 내일 14:00, 4인 미팅룸 조회 |
| `오늘 3시 포커스룸` | 오늘 15:00 포커스룸 조회 |
| `수정해줘` | 예약 수정 흐름 시작 |
| `빈 회의실 알려줘` | 현재 사용 가능한 회의실 조회 |

## 9. 예약 패널 설치 (/설치) — 채널 관리자용

채널에 예약 버튼 패널을 설치합니다.

### 사용 방법

1. 패널을 설치할 채널에서 `/설치` 입력
2. 예약 패널이 채널에 게시되고 자동으로 고정됩니다

### 패널 버튼 구성

| 버튼 | 기능 |
|------|------|
| 📝 예약하기 | 미팅룸/포커스룸 예약 모달 열기 |
| ✏️ 수정/취소 | 내 예약 수정 또는 취소 |
| 📢 신고/건의 | 불편사항 신고 |
| 🔁 정기회의 | 정기 회의 예약 |
| 📊 현황 | 예약 현황 조회 |

## 10. 자주 묻는 질문 (FAQ)

### Q. 다른 사람의 예약을 수정/취소할 수 있나요?
A. 아니요. 본인이 만든 예약만 수정/취소할 수 있습니다.

### Q. 예약하면 참석자에게 어떻게 알림이 가나요?
A. 구글 캘린더 초대장이 참석자 이메일로 자동 발송됩니다. 외부 게스트에게도 동일하게 발송됩니다.

### Q. 포커스룸은 참석자를 추가할 수 없나요?
A. 포커스룸은 1인 집중 업무 공간이므로 예약자 본인만 사용합니다.

### Q. 과거 날짜로 예약할 수 있나요?
A. 아니요. 현재 시각 이후의 시간만 예약할 수 있습니다.

### Q. 회의실이 다 차서 예약이 안 돼요.
A. 시간대를 변경하거나, 더 작은/큰 회의실을 선택해보세요. `/즉시예약`으로 현재 빈 회의실을 확인할 수도 있습니다.

### Q. 캘린더에서 직접 예약해도 되나요?
A. 빈방있소를 통해 예약하면 충돌 방지, 자동 알림, 기록 관리가 됩니다. 가능하면 봇을 통해 예약해주세요.

### Q. 봇이 응답하지 않아요.
A. 잠시 후 다시 시도해주세요. 계속 문제가 있으면 관리자에게 문의해주세요.

### Q. `/사용방법` 커맨드가 뭔가요?
A. 슬랙에서 `/사용방법`을 입력하면 빈방있소의 커맨드 목록과 간단한 안내를 볼 수 있습니다.
